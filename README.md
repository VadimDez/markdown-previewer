#Markdown previewer

FreeCodeCamp project based on [React](https://facebook.github.io/react/)

[Demo page](http://codepen.io/VadimDez/full/pgadjj/)

## Build

Install dependencies first

```
npm i
```

then build

```
./node_modules/webpack/bin/webpack.js 
```

## Run

Open index.html in browser
